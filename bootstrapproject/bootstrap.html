WHAT HAVE YOU LEARNED THIS SEMESTER?
Assignment # 1, was an assignment that required the class to construct a website using their new and prior knowledge that they learned in MMA/MMP 100. The assignment was to connect 4 pages together using css and html format. Each page of the website was expected to be about the options BMCC offers within the schools career programs.
Midterm Prep, was a section of the course where we took a few weeks to develop a sitemap, an outline, answer questions and used photoshop to create what imagined our website would look like.
The Midterm Assignment, was to create a music festival website. The website had to be creative as well as organized. We were expected to attract an audience and use the proper html and css code to develop a original site. By using css coding the class was expected to create a site that would convert to cellphone and tablet sizes.
Flexboxes, were a important part of the midterm prep and was introduced in this course. The boxes were a row that had an organized and creative way to keep info and pictures neat on your website.
